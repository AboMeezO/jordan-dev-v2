# Reminders

## Purpose
Allow users and admins to schedule timed notifications for themselves or the community.

## Glossary

### Reminder
A scheduled notification with a message, delivery time, and target (DM for personal, channel for community-wide).

### Personal Reminder
A reminder sent via DM to the user who created it. Only visible to that user.

### Community Reminder
A reminder posted to a Discord channel by an admin. Visible to all members of that channel.

### Natural-Language Time Parser
A feature that converts human-readable time expressions ("in 30 minutes", "next Monday at 3pm") into exact timestamps for scheduling.
